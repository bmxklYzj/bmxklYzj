代码

最后附上我实验的代码，可以在不同的浏览器中、手机上运行看看效果，你会发现随着你点击不同的input会呼出不同的键盘，这是多么有趣。
实验结果：

  1. 小米android5、google输入法、小米浏览器，点击date/time/month/week等不同input会呼出不同的原生组件，url、search、email键盘会有细微差别，但number会弹出类似tel的数字键盘和ios不一样。点击color浏览器会卡死。
  2. 小米android5、google输入法、chrome浏览器，和小米浏览器基本差不多，只是弹出的date/time/month/week是chrome自定义的界面（尤其是date弹出了自定义的日历，很漂亮啊），color使用正常。

结论：不同的设备对input type的支持程度和界面都有很大的不同，但是不支持的属性都会默认成text。
