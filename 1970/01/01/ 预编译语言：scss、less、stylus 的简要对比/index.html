预编译语言：scss、less、stylus 的简要对比

scss 是 sass 的升级版，没用过sass就不说了

先说历史

CSS 预处理器是一种语言用来为 CSS 增加一些编程的的特性，无需考虑浏览器的兼容性问题，例如你可以在 CSS 中使用变量、简单的程序逻辑、函数等等在编程语言中的一些基本技巧，可以让你的 CSS 更见简洁，适应性更强，代码更直观等诸多好处。

异同

强烈推荐：baidu efe 的一篇经典对比文章

github上一篇对比 scss 和 less 的文章，基本看一遍知识点都能回顾起来了

Less 的基本语法属于「CSS 风格」，而 Scss、Stylus 相比之下激进一些，利用缩进、空格和换行来减少需要输入的字符。不过区别在于 Scss、Stylus 同时也兼容「CSS 风格」代码

变量

三者在声明和使用的变量的时候有较大差异

less 是使用 @a:b
scss 是使用 $a:b
stylus 是使用 a=b
